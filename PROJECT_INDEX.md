# Project Index

Dokumen ini adalah peta cepat untuk memahami proyek penelitian ini tanpa harus membaca seluruh repo sekaligus.

## Tujuan Proyek

Repositori ini berisi bahan penelitian, referensi, catatan kerja, dan artefak penulisan untuk proyek ML/IoT CICIOT 2025.

## Urutan Baca yang Disarankan

1. `books/PLAN.md`
2. `references/gap-penelitian.md`
3. `books/timeline.md`
4. `logbook.md`
5. `references/references-paper.md`
6. `references/references-code.md`
7. `references/references-workflow.md`

## Isi Folder Utama

- `books/` berisi draft, rencana, timeline, brainstorming, proposal, dan dokumen penulisan.
- `references/` berisi gap penelitian dan kumpulan referensi pendukung.
- `research/` berisi notebook, skrip data, dan eksperimen teknis.
- `docs/` berisi dokumen pendukung dan teori penunjang.
- `assets/` berisi gambar, diagram, dan visual pendukung.

## File Penting

- `books/PLAN.md`: kerangka kerja penelitian.
- `books/timeline.md`: jadwal pengerjaan.
- `references/gap-penelitian.md`: alasan penelitian dan celah yang ingin diisi.
- `logbook.md`: catatan progres dan keputusan.
- `mindmap.mindmap`: gambaran hubungan ide.

## Cara Memperbarui

- Tambahkan entri baru jika ada file yang langsung memengaruhi arah penelitian.
- Ringkas, jangan panjang-panjang. Tujuannya navigasi, bukan dokumentasi detail.
- Kalau ada perubahan besar pada metode atau scope, update file ini lebih dulu daripada menyebar catatan ke banyak tempat.
