Dataset Comparison
CIC IoT 2023 · versus · CIC IIoT 2025 (DataSense)
CIC IoT
2023
Benchmark keamanan jaringan IoT umum. Topologi dengan 105 perangkat IoT nyata — smart home, office, campus. Baseline utama di banyak penelitian IDS berbasis trafik jaringan.
FOKUSIoT umum · smart environment
PERANGKAT105 perangkat IoT nyata
MODALITASTrafik jaringan (flow / packet)
SERANGAN33 jenis · 7 kategori
FITUR~48 fitur per flow
LABELBenign vs 33 attack labels
MASALAHClass imbalance 97.7%
CIC IIoT
2025
Benchmark Industrial IoT generasi terbaru. Menggabungkan sensor time-series dan trafik jaringan yang tersinkron — lebih dekat ke kondisi industri nyata, lebih kaya fitur.
FOKUSIndustrial IoT · sensor + jaringan
PERANGKAT40 perangkat IIoT beragam
MODALITASSensor time-series + trafik
SERANGAN50 jenis · 7 kategori
FITURMulti-fitur sensor + jaringan
LABELBenign vs 50 attack types
DESAINMulti-objektif: akurasi vs resource

Feature
Engineering
Raw traffic → variance threshold → correlation filter → 48-fitur subset. PCA untuk dimensionality reduction sebelum klasifikasi. Fokus pada efisiensi komputasi.
INPUTRaw packet capture
SELECTIONVariance + Correlation
REDUCTIONPCA · Subset
CLASSIFIERRandom Forest · SVM
Ensemble
Learning
Multiple base classifiers (RF, DT, SVM, NB) → majority voting atau stacking → ensemble decision. Imbalance ditangani dengan SMOTE atau class weighting.
BASERF · DT · SVM · NB
COMBINEVoting · Stacking
IMBALANCESMOTE · Class weight
EVALPrecision · Recall · F1
Deep
Learning
Normalisasi flow → LSTM/CNN untuk deteksi sequential. Autoencoder untuk anomaly detection pada traffic IoT terenkripsi atau tidak berlabel.
INPUTTime-series network flow
MODELLSTM · CNN · Autoencoder
TASKAnomaly · Classification
EDGEEncrypted traffic
Cross-
Validation
Stratified k-fold CV dengan per-class metrics. Beberapa studi menambahkan cross-dataset test untuk generalisasi ke environment lain.
EVALStratified k-fold CV
METRICSPrecision · Recall · F1
EXTRACross-dataset validation
REPORTConfusion matrix · ROC
